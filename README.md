# 🌍 Practice_Passport

📅 Date: 27-03-2026 - Initial setup and server testing  
📅 Date: 28-03-2026 - Implemented authentication system (Dashboard + Signup + Login + Redirect)  

💡 A full-stack web application focused on user authentication using Passport.js with signup, login, session handling, flash messages, EJS templating (ejs-mate), and MongoDB integration.  

🚀 Features:  
1. User Signup & Login (Passport.js)  
2. Flash Messages  
3. EJS Layouts (ejs-mate)  
4. MongoDB Integration  

🛠️ Tech Stack:  
1. Node.js  
2. Express.js  
3. MongoDB & Mongoose  
4. Passport.js  
5. EJS  
6. Bootstrap  

⚙️ Setup & Run:  
1️⃣ Clone Repository  
git clone https://github.com/your-username/your-repo-name.git  
cd your-repo-name  

2️⃣ Install Dependencies  
npm install  

3️⃣ Create .env file and add:  
DB_URL=mongodb://127.0.0.1:27017/wanderlust  
SESSION_SECRET=yourSecretKey  

4️⃣ Start MongoDB locally  

5️⃣ Run Application  
node app.js  

6️⃣ Open in Browser  
http://localhost:8080/dashboard
